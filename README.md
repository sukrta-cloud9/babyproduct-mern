# BabyBay 

BabyBay is a full-stack MERN e-commerce application for baby products.

Users can browse products, create an account, log in, add products to their wishlist and cart, update quantities, and manage their shopping experience.

Administrators can upload new products along with product images.


## Live Demo

Frontend:
https://babyproduct-mern.vercel.app/

Backend:
https://babyproduct-mern.onrender.com

## Features

- User Registration & Login
- Product Listing
- Product Details Page
- Wishlist
- Shopping Cart
- Quantity Update
- Remove Cart Items
- Admin Product Upload
- Image Upload using Multer
- MongoDB Database
- Responsive Design
